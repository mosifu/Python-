# 字面量
程序中，直接书写的固定值（数据），称为字面量  

| 类型            | 说明                         | 示例                           |
|---------------|----------------------------|------------------------------|
| 数字类型          | 整型(int) <br> 浮点数/小数(float) | 10 18 -5 0 <br> 3.2 -4.5 1.0 |
| 布尔(bool)      | 表达生活中的逻辑>，真或假              | True False                   |
| 字符串(str)      | 描述文本的一种数据类型                | '人生大事,Python'                |
| 空值(None Type) | 表示空或无值,仅包含一个值None          | None                         |
| 数据容器          | 存储多项数据的容器                  | 列表/元组/集合/字典                  |
