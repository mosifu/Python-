#此项目用来记录Python学习过程,利用cursor+Claude code+deepseek v4.
视频课程参照:- **课程名称**：黑马程序员 Python+AI 零基础入门到大神全套视频课程
              - **B站链接**：https://www.bilibili.com/video/BV1sHU9BmEne/
主分支创建了一个skill和两个agent.
skill来规划学习路线,以及学习笔记生成
learning-analyst agent用来对每日学习笔记以及代码进行分析与总结,并给出建议和练习题
practice-reviewer agent针对学习分析agent给出的题目进行批改
