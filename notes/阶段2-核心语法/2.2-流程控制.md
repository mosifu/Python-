## if 条件判断

> 📺 **对应课程**：第 22–25 集

### 1. 什么是 if

- **是什么**：if 就像十字路口的红绿灯 — 满足条件走绿灯（执行代码），不满足走红灯（跳过代码）。
- **Java 对比**：
  - 相似点：逻辑完全一致，`if`/`else if`/`else` 语义相同
  - 不同点：Python **用缩进代替 `{}`**、`elif` 代替 `else if`、条件**不需要括号**（写了也行但不推荐）

### 2. 基本语法

```python
# 单分支 — 没有花括号！
score = 689
if score > 680:
    print("我去读清华")      # 缩进 = Java 的 { }
    print("超过680的都来读")  # 同级缩进 = 同一代码块

# 双分支 — else 后面加冒号
age = 17
if age >= 18:
    print("已成年")
else:
    print("未成年")

# 多分支 — elif 不是 else if！
score = 85
if score >= 85:
    print("优秀")
elif score >= 60:      # ← Java 是 else if，Python 是 elif
    print("及格")
else:
    print("不及格")
```

**Java 对比速查**：

| | Python | Java |
|------|--------|------|
| 条件括号 | 可省略 `if x > 0:` | 必须 `if (x > 0)` |
| 代码块 | 缩进（冒号+缩进） | `{ }` 花括号 |
| 多分支 | `elif` | `else if` |
| 句尾 | 无分号 | `;` |
| 空语句 | `pass` | `;`（空分号） |

### 3. 实战案例

```python
# 案例1：登录验证
right_account = "188888888"
right_password = "6666666"
account = input("请输入账号：")
password = input("请输入密码：")

if account == right_account and password == right_password:
    print("登录成功！")
else:
    print("账号或密码错误！")

# 案例2：闰年判断（非整百年能被4整除，整百年能被400整除）
year = int(input("请输入年份："))
if (year % 100 != 0 and year % 4 == 0) or year % 400 == 0:
    print(f"{year}是闰年")
else:
    print(f"{year}是平年")

# 案例3：多用户登录（多分支）
if account == "admin" and password == "666888":
    print("admin 登录成功")
elif account == "root" and password == "12345":
    print("root 登录成功")
elif account == "mosifu" and password == "666":
    print("mosifu 登录成功")
else:
    print("用户名或密码错误")
```

### 4. 嵌套 if

```python
# 案例：三角形类型判断
a, b, c = 3, 3, 5

# 先判断能否构成三角形
if a + b > c and a + c > b and b + c > a:
    # 能构成，再细分类型
    if a == b == c:
        print("等边三角形")
    elif a == b or b == c or a == c:
        print("等腰三角形")
    else:
        print("普通三角形")
else:
    print("不能构成三角形")
```

### 5. `pass` 语句 — 占位符

```python
# pass = 什么都不做，相当于 Java 的空分号 ;
# 用在语法需要代码但逻辑暂不实现的地方

if condition:
    pass  # TODO: 待实现
else:
    print("已处理")
```

## match 模式匹配

> 📺 **对应课程**：第 26–27 集
> ⚠️ **需要 Python 3.10+**

### 1. 什么是 match

- **是什么**：match 像快递分拣机 — 包裹（值）来了，按标签（模式）自动分配到对应的传送带（代码块）。
- **Java 对比**：
  - 相似点：≈ Java 14+ 的 `switch` 表达式 + 模式匹配
  - 不同点：Python match **不需要 break**（不会穿透）；功能更强（支持 `|`、`if` 守卫、序列解包）

### 2. 基本语法

```python
# match...case — Python 的 switch
day = input("请输入星期几(1-7)：")
match day:
    case "1":
        print("星期一，上班日")
    case "2":
        print("星期二，项目日")
    case "3":
        print("星期三，研发日")
    case "4":
        print("星期四，研讨日")
    case "5":
        print("星期五，摸鱼日")
    case "6" | "7":        # ← | = OR，匹配多个值
        print("周末，休息日")
    case _:                # ← _ = default，匹配所有
        print("输入日期不存在")
```

**Java 对比**：

| | Python match | Java switch |
|------|-------------|-------------|
| 关键字 | `match` / `case` | `switch` / `case` |
| break | **不需要**，自动不穿透 | 必须写 `break`，否则穿透 |
| default | `case _:` | `default:` |
| 多值 | `case "6" \| "7":` | `case "6": case "7":` (穿透) |
| 守卫 | `case "/" if b != 0:` | ❌ 无原生支持 |

### 3. case 守卫（Guard）— `if` 条件附加

```python
# case 后面加 if 条件，只有值和条件同时满足才匹配
a, b = 10, 0
operator = "/"

match operator:
    case "+":
        print(f"{a} + {b} = {a + b}")
    case "-":
        print(f"{a} - {b} = {a - b}")
    case "*":
        print(f"{a} * {b} = {a * b}")
    case "/" if b != 0:   # ← 守卫：防止除零！
        print(f"{a} / {b} = {a / b}")
    case "/":              # b == 0 走这里
        print("除数不能为 0！")
    case _:
        print(f"未知运算符 {operator}")
```

> 💡 `if b != 0` 在 `case` 后面叫 **守卫（Guard）**，只有模式匹配 **且** 守卫条件为 `True` 时才执行。Java 的 `switch` 做不到这个。

### 4. 综合案例：游戏角色控制

```python
move = input("请输入指令：")
match move:
    case "w" | "W" | "上":
        print("角色向上移动")
    case "s" | "S" | "下":
        print("角色向下移动")
    case "a" | "A" | "左":
        print("角色向左移动")
    case "d" | "D" | "右":
        print("角色向右移动")
    case " " | "跳":
        print("角色跳跃")
    case "j" | "J" | "攻击":
        print("角色发动攻击")
    case "esc" | "ESC" | "退出":
        print("退出游戏")
    case _:
        print("无效指令")
```

## if vs match：什么时候用哪个

| 场景 | 推荐 | 原因 |
|------|:--:|------|
| 范围判断（> <） | `if` | match 不适合范围 |
| 精确值匹配（==） | `match` | 代码更清晰 |
| 复杂多条件 | `if` | if 灵活度更高 |
| 少量分支（2-3） | `if` | match 杀鸡用牛刀 |
| 多种输入映射 | `match` | `\|` 多值匹配很优雅 |

---

## 常见坑点（Java 背景）

1. **`elif` 不是 `else if`** — 写 `else if` 会报 SyntaxError
2. **忘了冒号 `:`** — `if x > 0` 报错，必须 `if x > 0:`
3. **缩进不一致** — 同级代码必须对齐，混用空格和 Tab 报错
4. **match 不需要 break** — 每个 case 自动独立，不像 Java switch 会穿透
5. **match 是 Python 3.10+** — 旧版本不支持的语法
6. **条件链写法** — `60 <= score < 85` Python 支持，Java 必须 `score >= 60 && score < 85`

---

## 小结（上篇：if + match）

1. `if`/`elif`/`else` 逻辑和 Java 一样，语法不同：**无括号、无花括号、elif 代替 else if、冒号+缩进**
2. `match`/`case` 是 Python 3.10+ 的 switch 升级版：**不穿透、支持 `|` 多值、支持 `if` 守卫、`_` 通配**
3. 范围判断用 `if`，精确值匹配多用 `match`；`pass` 是空语句占位符

## while 循环

> 📺 **对应课程**：第 28–30 集

### 1. 什么是 while

- **是什么**：while 就像自动洗衣机 — "只要还没洗干净（条件成立），就不停地转（执行循环体）"，直到条件不满足才停。
- **Java 对比**：
  - 相似点：`while` 语法和逻辑完全一样
  - 不同点：Python 有 `while...else`（Java 没有）；条件同样不需要括号

### 2. 基本语法

```python
# 打印 10 遍 "人生苦短，我用Python"
i = 0
while i < 10:
    i += 1
    print(f"第{i}遍：人生苦短，我用Python！")
```

### 3. while...else — Python 独有！

```python
# else 在循环正常结束（非 break）时执行
i = 0
while i < 10:
    i += 1
    print(f"第{i}遍")
else:
    print("循环正常结束！")  # 循环条件为 False 时执行

# 如果被 break 打断，else 不会执行
while True:
    if 条件满足:
        break   # ← else 不执行
else:
    print("这行不会输出")
```

> 💡 **Java 没有 `while...else`**，这在 Python 中常用于"循环搜索某值，未找到时执行 else"的场景。

### 4. 实战案例

```python
# 案例：计算 1-100 所有偶数的和
i = 1
total = 0
while i <= 100:
    if i % 2 == 0:
        total += i
    i += 1
print(f"1-100 偶数和: {total}")
```

## for 循环 & range

> 📺 **对应课程**：第 31–33 集

### 1. 什么是 for

- **是什么**：for 就像点名 — "花名册上有谁，就点到谁"，一个一个来，每个都处理一遍。
- **Java 对比**：
  - 相似点：≈ Java 的增强 for（`for (String s : list)`）
  - 不同点：Python 的 `for` **没有传统的 `for (int i=0; i<n; i++)` 三件套**，用 `range()` 实现；也有 `for...else`

### 2. 基本语法

```python
# 遍历字符串 — 逐个字符
for ch in "Python":
    print(ch)   # P y t h o n

# 遍历列表
for item in ["苹果", "香蕉", "橘子"]:
    print(item)
```

**Java 对比**：

| Python | Java |
|--------|------|
| `for i in range(10):` | `for (int i=0; i<10; i++)` |
| `for item in list:` | `for (String item : list)` |
| `for ch in "hello":` | `for (char c : "hello".toCharArray())` |

### 3. `range()` — Python 的计次器

```python
# range(end) — 从 0 到 end-1
for i in range(5):
    print(i, end=" ")  # 0 1 2 3 4

# range(start, end) — 从 start 到 end-1
for i in range(3, 7):
    print(i, end=" ")  # 3 4 5 6

# range(start, end, step) — 带步长
for i in range(1, 10, 2):
    print(i, end=" ")  # 1 3 5 7 9
```

> 💡 `range()` 生成的数字序列**不包含 end**，和 Python 切片逻辑一致。

### 4. for...else — Python 独有！

```python
for i in "hello":
    print(i)
else:
    print("遍历结束~")  # 循环正常结束（非 break）时执行
```

### 5. 实战案例

```python
# 案例1：计算 1-100 所有奇数之和（方法一：步长2）
total = 0
for i in range(1, 101, 2):
    total += i
print(f"1-100 奇数和: {total}")

# 案例2：计算 100-500 之间所有 3 的倍数之和
total = 0
for i in range(100, 501):
    if i % 3 == 0:
        total += i
print(f"100-500 3的倍数和: {total}")
```

### 6. while vs for 选型指南

| 场景 | 推荐 | 原因 |
|------|:--:|------|
| 知道循环次数 | `for` | range() 直接控制次数 |
| 循环次数未知 | `while` | 条件控制更灵活 |
| 遍历容器 | `for` | 语法更简洁 |
| 登录重试 | `while` | 不知道几次能成功 |
| 无限循环 | `while True` | for 不适合 |

## break & continue

> 📺 **对应课程**：第 34 集

- **Java 对比**：语义完全一样，无差异。

| 关键字 | 作用 | 影响 else？ |
|--------|------|:--:|
| `break` | **跳出整个循环** | ✅ else 不执行 |
| `continue` | **跳过本次**，进入下一次 | ❌ else 仍然执行 |

```python
# 登录案例：while True + break + continue
while True:
    username = input("用户名: ")
    password = input("密码: ")

    # continue：输入为空则跳过本次
    if username == "" or password == "":
        print("不能为空！")
        continue   # ← 回到 while True 重新输入

    # break：登录成功跳出循环
    if username == "admin" and password == "666888":
        print("登录成功！")
        break      # ← 跳出循环，else 不执行

    print("用户名或密码错误！")
```

## 嵌套循环

> 📺 **对应课程**：第 35 集

### 1. 什么是嵌套循环

- **是什么**：循环里面套循环，像钟表的时针和分针 — 分针转一圈（内循环），时针才走一格（外循环）。

### 2. 核心要点

```python
# 打印 m×n 的长方形
m, n = 5, 3
for i in range(n):          # 外循环：控制行
    for j in range(m):      # 内循环：控制列
        print("*", end="")  # end="" 不换行
    print()                 # 一行结束换行
```

> ⚠️ **关键**：`print("*", end="")` 的 `end=""` 让星号不换行，内循环完成后单独 `print()` 换行。

### 3. 经典案例

```python
# 九九乘法表
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f"{i}×{j}={i*j}", end="\t")
    print()

# 数字金字塔
for i in range(1, 6):
    for j in range(1, i + 1):
        print(j, end=" ")
    print()

# 国际象棋棋盘（黑白格交替）
n = 8
for row in range(n):
    for col in range(n):
        if (row + col) % 2 == 0:
            print("■", end="")
        else:
            print("□", end="")
    print()
```

## 综合案例

### 案例 1：猜数字游戏

集合了 `random`、`while True`、`if/elif/else`、`break`、`input()`、`int()` 等几乎所有已学知识：

```python
import random

target = random.randint(1, 100)  # 1-100 随机数
while True:
    guess = int(input("请输入预测的数字: "))
    if guess > target:
        print("猜大了！")
    elif guess < target:
        print("猜小了！")
    else:
        print("猜对了~")
        break
print(f"随机数字为: {target}")
```

### 案例 2：for + 条件筛选

```python
# 计算 1-1000 之间所有 5 的倍数的和
total = 0
for i in range(1, 1001):
    if i % 5 == 0:
        total += i
print(f"1-1000 所有5的倍数之和: {total}")
```

### 案例 3：for 遍历字符串 + 计数

```python
# 统计用户输入字符串中 'a' 和 'k' 的出现次数
text = input("请输入随机的字符串: ")
count_a = count_k = 0       # 链式赋值初始化两个计数器

for ch in text:
    if ch == 'a':
        count_a += 1
    elif ch == 'k':
        count_k += 1

print(f"字符串 '{text}' 中有 {count_a} 个 'a'，{count_k} 个 'k'")
```

> 💡 **技巧**：`count_a = count_k = 0` 是 Python 的链式赋值，一行初始化两个变量；`elif` 比两个独立的 `if` 更高效（一个字符不可能同时是 'a' 又是 'k'）。

## 常见坑点（下篇：循环）

14. **`while` 忘记更新条件变量** — 写成死循环，检查循环体里是否改了条件变量
15. **`range()` 不包含 end** — `range(1, 10)` 只到 9，和 Java `for(int i=1; i<10; i++)` 对应
16. **`print()` 默认换行** — 想在同一行输出必须加 `end=""`
17. **`while...else` 和 `for...else`** — Java 没有这个语法，break 跳出后 else 不执行
18. **Python 没有 `do-while`** — 用 `while True + break` 模拟
19. **嵌套循环的外层变量命名** — 混淆 `i`/`j` 容易写错，建议用语义化变量 `row`/`col`

---

## 小结

1. `while` 条件循环，和 Java 一样；Python 多了 `while...else`
2. `for` + `range()` 替代 Java 的 `for(;;)` 三件套；`for...else` Java 无此语法
3. `break` 跳出循环（else 不执行），`continue` 跳过本次（else 仍执行）
4. 嵌套循环：外层控行、内层控列，`end=""` 控制不换行
