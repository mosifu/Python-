## 函数基础

> 📺 **对应课程**：第 57–63 集

### 1. 什么是函数

- **是什么**：函数就像一个菜谱——给它食材（参数），按步骤做（函数体），端出成品（返回值）。写一次，到处用。
- **Java 对比**：
  - 相似点：都是封装可复用逻辑
  - 不同点：Python 函数是**一等公民**（可赋值给变量、当参数传递）；不用写 `public static`；函数可独立存在，不必属于某个类

```python
# 定义 — def + 冒号 + 缩进
def greet():
    print("hello world")
    print("------------")

# 调用
greet()
```

| | Python | Java |
|------|--------|------|
| 定义 | `def func():` | `public static void func() { }` |
| 块 | 缩进 | `{ }` |
| 独立存在 | ✅ 函数可以不在类里 | ❌ 必须在类里（方法） |
| 返回多值 | `return a, b` → 元组 | ❌ 只能单一返回值 |

> ⚠️ **必须先定义后调用**，和 Java 不同（Java 方法顺序随意）。

### 2. 参数

```python
# 单参数
def circle_area(r):
    return 3.14 * r ** 2

# 多参数 — 逗号隔开
def rectangle_area(length, width):
    return length * width

print(circle_area(5))           # 78.5
print(rectangle_area(4, 3))     # 12
```

### 3. 返回值 — 多值返回（Python 独有！）

```python
# 返回多个值 → 自动打包为元组
def circle_info(r):
    area = 3.14 * r ** 2
    perimeter = 2 * 3.14 * r
    return round(area, 1), round(perimeter, 1)   # 返回元组！

# 方式1：一个变量接收 → 元组
result = circle_info(5)
print(result)         # (78.5, 31.4)
print(type(result))   # <class 'tuple'>

# 方式2：解包接收（推荐！）
area, perimeter = circle_info(5)
print(f"面积: {area}, 周长: {perimeter}")
```

> 💡 Java 只能返回一个值，Python 靠元组轻松返回多个值——这是日常开发中的常用模式。

### 4. 函数嵌套调用

```python
def a():
    print("a...before")
    b()
    print("a...after")

def b():
    print("b...before")
    c()
    print("b...after")

def c():
    print("c...")

a()
# 输出：a...before → b...before → c... → b...after → a...after
# 执行顺序像栈：后进先出（LIFO）
```

### 5. Docstring — 函数文档

```python
def triangle_area(base, height):
    """
    根据传入的底和高计算三角形面积
    :param base:  底
    :param height: 高
    :return: 面积
    """
    return round(base * height / 2, 1)
```

## 函数进阶

> 📺 **对应课程**：第 64–70 集

### 1. 变量作用域

```python
num = 100  # 全局变量 — 整个文件都能访问

def calc(r):
    pi = 3.14     # 局部变量 — 只在函数内有效
    global num    # 声明要修改全局变量（放在函数顶部）
    num = 10000   # 修改的是全局 num
    return pi * r ** 2

print(calc(10))   # 314.0
print(num)        # 10000（全局变量被修改了）
```

| 关键字 | 作用 | 使用场景 |
|--------|------|------|
| `global` | 函数内修改全局变量 | 程序状态、配置、计数器 |

> ⚠️ `global` 声明应放在函数体**最前面**（PEP 8）。无 `global` 时，函数内同名变量会创建新的局部变量，不影响全局。

### 2. 传参方式

```python
def register(name, age, gender, city):
    return {"name": name, "age": age, "gender": gender, "city": city}

# 方式1：位置传参 — 按顺序对应
register("mosifu", 23, "男", "贵阳")

# 方式2：关键字传参 — 顺序随意
register(name="lan", gender="女", city="贵阳", age=23)

# 方式3：混合传参 — 位置必须在关键字前面！
register("jian", 23, city="贵阳", gender="女")
# register(name="jian", 23, city="贵阳")  # ❌ 关键字后不能再跟位置参数
```

### 3. 默认参数

```python
# 默认参数必须放在非默认参数的后面
def register(name, age, gender="男", city="东莞"):
    return {"name": name, "age": age, "gender": gender, "city": city}

register("mo", 23)                        # gender和city用默认值
register("lan", 24, "女")                  # city用默认值
register("momo", 25, city="贵阳")          # gender用默认值
```

**Java 对比**：≈ 方法重载 `register(name, age)` + `register(name, age, gender)` + `register(name, age, gender, city)`，Python 一个函数搞定。

### 4. 不定长参数

**`*args` — 接收任意个位置参数 → 元组**：

```python
# 计算任意个数的最大值、最小值、平均值
def calc_data(*args):
    return min(args), max(args), round(sum(args) / len(args), 2)

m, a, v = calc_data(1, 3, 2, 45, 5, 2, 4)
print(f"最大值:{a}, 最小值:{m}, 平均值:{v}")
```

**`**kwargs` — 接收任意个关键字参数 → 字典**：

```python
def calc_data(*args, **kwargs):
    min_val = min(args)
    max_val = max(args)
    avg_val = sum(args) / len(args)

    # kwargs 控制可选行为
    if kwargs.get("round") is not None:
        avg_val = round(avg_val, kwargs["round"])
    if kwargs.get("print"):
        print(f"最大值:{max_val}, 最小值:{min_val}, 平均值:{avg_val}")

    return min_val, max_val, avg_val

# round=2 控制精度, print=True 控制是否打印
calc_data(1, 3, 2, 45, 5, 2, 4, round=2, print=True)
calc_data(13, 23, 42, 45, 25, 42, 4)         # 不打印，不取整
```

**Java 对比**：

| Python | Java |
|--------|------|
| `*args` | `void func(int... args)`（可变参数） |
| `**kwargs` | ❌ 无直接语法，通常用 Map 或 Builder 模式 |

### 5. 函数作为参数（高阶函数）

```python
# Python 中函数是一等公民 — 可以当参数传递！
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def calc(x, y, operation):
    return operation(x, y)   # operation 是一个函数

print(calc(2, 3, add))       # 5
print(calc(2, 3, subtract))  # -1
```

> 💡 **Java 对比**：≈ 函数式接口 + Lambda，如 `BiFunction<Integer, Integer, Integer>`

### 6. lambda 匿名函数

```python
# 命名函数
def add(a, b):
    return a + b

# lambda 单行匿名函数 — lambda 参数: 返回值
add_lambda = lambda a, b: a + b
print(add_lambda(10, 20))  # 30

# 最常用场景：sort 的自定义排序 key
data = ["C++", "C", "Python", "Java", "Go", "JavaScript", "Rust"]
data.sort(key=lambda item: len(item))  # 按字符串长度排序
print(data)  # ['C', 'Go', 'C++', 'Java', 'Rust', 'Python', 'JavaScript']
```

| | `def` 命名函数 | `lambda` 匿名函数 |
|------|:--:|:--:|
| 代码量 | 多行 | 只能一行表达式 |
| 可复用 | ✅ | ❌ 用完即弃 |
| 典型场景 | 通用逻辑封装 | sort 的 key、filter 的条件 |
| 可含 print | ✅ | ❌ 不推荐（lambda 应无副作用） |

### 7. 递归函数

```python
# 阶乘：n! = n * (n-1) * ... * 1
def factorial(n):
    """计算 n 的阶乘（递归实现）"""
    if n == 1:          # 终结点 — 必须有！
        return 1
    return n * factorial(n - 1)

print(factorial(4))  # 24
# 执行过程：factorial(4) = 4 * factorial(3) = 4 * 6 = 24
#          factorial(3) = 3 * factorial(2) = 3 * 2 = 6
#          factorial(2) = 2 * factorial(1) = 2 * 1 = 2
#          factorial(1) = 1（终结点）
```

> ⚠️ 递归**必须要有终结点**，否则无限递归 → `RecursionError`。

---

## 常见坑点

1. **`def` 内变量是局部变量** — 函数内赋值不写 `global`，不会影响外部同名变量
2. **默认参数不要用可变对象** — `def f(lst=[])` 是经典陷阱（之后细讲）
3. **位置参数必须在关键字参数前** — `f(name="x", 23)` 报 SyntaxError
4. **不要用 `sum`/`list`/`str` 等内置函数名做变量名** — 覆盖内置函数，之后调用报错
5. **`global` 声明放函数最前面** — PEP 8 推荐
6. **递归必须有终结点** — 忘了就 `RecursionError`

## 综合案例：电商订单计算器

> 💡 来自学习者的实战练习，串联了 `*args`/默认参数/list 推导式/多条件判断

```python
def calc_order_cost(*args, coupon=0, score=0, express=0.0):
    """
    计算电商订单总金额
    :param args: 商品元组 (商品名, 单价, 数量)
    :param coupon: 优惠券金额（满5000可用，不超过商品总价）
    :param score: 积分（满5000可用，100积分=1元，整百抵扣）
    :param express: 运费
    :return: 订单总金额
    """
    # 1. 计算商品总金额
    prices = [goods[1] * goods[2] for goods in args]
    total = sum(prices)

    # 2. 扣减优惠券（满5000可用，不超总价）
    if total >= 5000 and coupon <= total:
        total -= coupon

    # 3. 扣减积分（满5000可用，不超总价，整百抵扣）
    if total >= 5000 and score // 100 <= total:
        total -= score // 100

    # 4. 加运费
    total += express
    return total

# 调用：手机4999×2 + 茶叶78×5，优惠券10，积分4000（抵扣40元），运费9.9
result = calc_order_cost(
    ("手机", 4999, 2), ("茶叶", 78, 5),
    coupon=10, score=4000, express=9.9
)
print(result)  # 10347.9
```

> 📊 **计算步骤**：商品总价 10388 → 减优惠券 10 = 10378 → 减积分 40 = 10338 → 加运费 9.9 = **10347.9**

## 小结

1. **函数基础**：`def` 定义 → 参数传递 → `return` 多值（元组）→ 解包接收；必须先定义后调用
2. **传参方式**：位置/关键字/混合/默认参数/`*args`（元组）/`**kwargs`（字典）
3. **进阶特性**：函数是一等公民可作参数、`lambda` 单行匿名函数常用于 `sort(key=)`、递归必须有终结点
