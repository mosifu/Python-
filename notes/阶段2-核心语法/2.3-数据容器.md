## list 列表

> 📺 **对应课程**：第 36–42 集

### 1. 什么是 list

- **是什么**：列表就像火车 — 每节车厢可以装不同类型的东西，车厢编号从 0 开始。
- **Java 对比**：
  - 相似点：≈ `ArrayList`，有序、可变、可索引
  - 不同点：Python 列表**可存不同类型**；语法直接用 `[]`，无需 `new`

```python
# 定义 — 比 Java 简洁得多
s = [1, 3, 4.5, 'hello', True]   # 混合类型完全 OK
print(type(s))  # <class 'list'>
```

| | Python list | Java ArrayList |
|------|-------------|---------------|
| 定义 | `[1, 2, 3]` | `new ArrayList<>(List.of(1,2,3))` |
| 混合类型 | ✅ 可以 | ❌ 必须同类型（泛型） |
| 索引 | `s[0]` `s[-1]` | `s.get(0)` |
| 修改 | `s[0] = 10` | `s.set(0, 10)` |

### 2. 索引与切片

```python
h = ["A", "B", "C", "D", "E", "F", "G", "H", "I", "J"]

# 索引 — 正向从 0，反向从 -1
print(h[0])    # A
print(h[-1])   # J（最后一个）
print(h[-5])   # F（倒数第5个）

# 切片 — list[start:end:step]
print(h[0:6])     # ['A','B','C','D','E','F']  end不包含
print(h[:4])      # ['A','B','C','D']  省略start=从头
print(h[1:-1:2])  # ['B','D','F','H']  步长为2
print(h[::-1])    # ['J','I',...,'A']  step=-1=反转
```

> 💡 切片规则和 `range()` 一样：**不包含 end**，支持反向步长。

### 3. 增删改查

| 操作 | 方法 | 说明 | Java 类比 |
|------|------|------|----------|
| **追加** | `.append(78)` | 末尾加一个元素 | `list.add(78)` |
| **插入** | `.insert(1, 78)` | 在索引1前插入 | `list.add(1, 78)` |
| **删除(值)** | `.remove(78)` | 删第一个匹配值 | `list.remove(Integer.valueOf(78))` |
| **删除(索引)** | `.pop(1)` | 删索引元素并返回 | `list.remove(1)` |
| **删除(索引)** | `del s[2]` | 纯删除，不返回 | 无直接对应 |
| **排序** | `.sort()` | 原地排序 | `Collections.sort(list)` |
| **反转** | `.reverse()` | 原地反转 | `Collections.reverse(list)` |
| **查索引** | `.index(x)` | 找第一个匹配 | `list.indexOf(x)` |
| **统计** | `.count(x)` | 计数 | `Collections.frequency(list, x)` |

```python
g = [56, 234, 634, 23, 63, 74, 211, 5]

g.append(78)       # [56, 234, 634, 23, 63, 74, 211, 5, 78]
g.insert(1, 99)    # [56, 99, 234, ..., 78]    在索引1前插入
g.remove(78)       # [56, 99, 234, ..., 5]     删第一个78
g.pop(1)           # [56, 234, ..., 5]         删索引1
g.pop()            # [56, 234, ..., 211]       无参数=删最后一个
g.sort()           # [5, 23, 56, 63, 74, 211, 234]
g.reverse()        # [234, 211, 74, 63, 56, 23, 5]
```

> ⚠️ `.sort()` 要求**元素类型相同**（不能混排 str 和 int）。
> ⚠️ `.sort()` 是**原地修改**返回 `None`，不是返回新列表。

### 4. 列表合并

```python
# 方式1：+ 拼接（Java 也可以用 addAll）
new_list = [1, 2, 3] + [4, 5, 6]     # [1, 2, 3, 4, 5, 6]

# 方式2：解包（Python 独有）
new_list = [*list1, *list2]

# 方式3：extend
list1.extend(list2)
```

### 5. 列表去重

```python
# 经典去重法：遍历 + not in
origin = [12, 24, 35, 12, 535, 125, 3, 24]
unique = []
for item in origin:
    if item not in unique:
        unique.append(item)

# 快捷法：set 去重（学到 set 时会讲）
unique = list(set(origin))  # 但会打乱顺序
```

### 6. 列表推导式（List Comprehension）

- **是什么**：用一行代码生成列表，就像数学里的集合推导 `{x² | x∈N, x<10}`。
- **Java 对比**：≈ Stream API `map().filter().collect()`

```python
# 语法1：[表达式 for i in 序列]
squares = [i**2 for i in range(1, 21)]     # 1-20的平方
# Java: IntStream.range(1,21).map(i->i*i).boxed().collect(toList())

# 语法2：[表达式 for i in 序列 if 条件]
evens = [i for i in range(1, 31) if i % 2 == 0]          # 偶数
squares_even = [i**2 for i in range(1, 31) if i % 2 == 0] # 偶数的平方

# 复杂筛选：能被3或5整除的数的平方
result = [i**2 for i in range(1, 31) if i % 3 == 0 or i % 5 == 0]

# 提取正数
positives = [i for i in [11, -21, 1231, -34, 10] if i > 0]
```

## str 字符串操作

> 📺 **对应课程**：第 43–45 集

### 1. str 的三大特性

| 特性 | 含义 | Java 对比 |
|------|------|----------|
| **不可变** | 一旦创建，内容不能修改 | ≈ `String` 的不可变性 |
| **有序** | 有索引，可切片 | ✅ 一样 |
| **可迭代** | 能用 `for` 遍历 | ✅ 一样 |

```python
s = "Hello-Python"
# s[0] = "h"       # ❌ TypeError: str 不可修改！

s = s.replace("H", "h")  # ✅ 返回新字符串，原字符串不变
```

> 💡 Python 无 `char` 类型，单个字符 `'a'` 也是 `str`。

### 2. 索引与切片

```python
str1 = "Hello-Python"

print(str1[0])       # H
print(str1[-1])      # n
print(str1[0:5])     # Hello
print(str1[::2])     # HloPto（隔一个取一个）
print(str1[::-1])    # nohtyP-olleH（反转）

# 负步长切片 — 开始索引必须在结束索引**右边**
print(str1[-8:-13:-1])  # oP-oll  从-8倒着走到-13
# print(str1[0:5:-1])   # 空！start=0, end=5, step=-1 → 方向不对
```

> ⚠️ **负步长坑**：`step` 为负时，start 必须在 end **右边**（索引更大的一方），否则结果为空。

### 3. 常用方法速查表

| 方法 | 作用 | 示例 | Java 类比 |
|------|------|------|----------|
| `.find("x")` | 找第一个匹配索引，没找到返回 -1 | `"hello".find("e")` → 1 | `indexOf()` |
| `.count("x")` | 统计出现次数 | `"hello".count("l")` → 2 | 用循环统计 |
| `.split("-")` | 按分隔符切割成**列表** | `"a-b-c".split("-")` → `['a','b','c']` | `split()` |
| `.upper()` | 全转大写 | `"hello".upper()` → `"HELLO"` | `toUpperCase()` |
| `.lower()` | 全转小写 | `"Hello".lower()` → `"hello"` | `toLowerCase()` |
| `.strip()` | 去两端空格 | `" hi ".strip()` → `"hi"` | `trim()` |
| `.replace(a, b)` | 替换子串 | `"a-b".replace("-", "_")` | `replace()` |
| `.startswith("x")` | 以 x 开头？ | `"hi".startswith("h")` → True | `startsWith()` |
| `.endswith("x")` | 以 x 结尾？ | `"hi".endswith("i")` → True | `endsWith()` |

```python
str2 = "  Hello-Python-Hello-World  "

print(str2.find("Hello"))     # 2（第一个Hello的位置）
print(str2.count("Hello"))    # 2（出现了2次）
print(str2.split('-'))        # ['  Hello', 'Python', 'Hello', 'World  ']
print(str2.upper())           # '  HELLO-PYTHON-HELLO-WORLD  '
print(str2.strip())           # 'Hello-Python-Hello-World'
print(str2.replace("-", "_")) # '  Hello_Python_Hello_World  '
print(str2)                   # 原始字符串没变！(不可变性)
```

### 4. 实战案例

```python
# 案例1：邮箱格式验证
email = input("请输入邮箱：")
if email.count("@") == 1 and "." in email:
    print("邮箱格式正确")
else:
    print("邮箱格式错误")

# 案例2：回文判断（正反读一样）
text = input("请输入字符串：")
half = len(text) // 2          # 整除取一半长度
if text[:half] == text[-1:-half-1:-1]:
    print(f"'{text}' 是回文！")
else:
    print(f"'{text}' 不是回文")

# 案例3：反转 + 大写
text = "hello"
reversed_upper = text[::-1].upper()  # OLLEH
```

## 常见坑点

1. **`.sort()` 返回 None** — 它是原地排序，别写 `new = list.sort()`
2. **list 越界** — 修改索引超出范围报 `IndexError`，切片越界不报错只返回空
3. **str 不可修改** — `s[0] = 'x'` 报 `TypeError`，用 `replace()` 或切片拼接
4. **`%` 和 `//` 混淆** — `%` 取余、`//` 整除，回文判断用 `//`
5. **负步长方向** — `[0:5:-1]` 返回空，start 必须在 end 右边
6. **列表推导式可读性** — 超过 2 层嵌套或复杂条件时，拆成普通 for 循环更清晰

---

## 小结

1. **list**：有序可变容器，索引 `[0]`/`[-1]`，切片 `[:]`，常用 `append/sort/pop`，推导式 `[x for x in list if]` 替代 for+append
2. **str**：有序不可变序列，`find/count/split/upper/strip/replace` 返回新值，原字符串不变
3. 切片三要素 `[start:end:step]`，end 不包含，负步长需方向一致

## tuple 元组

> 📺 **对应课程**：第 47–48 集

### 1. 什么是 tuple

- **是什么**：元组就像刻在石头上的清单——写上去就不能改，只能看。
- **Java 对比**：
  - 相似点：≈ 不可变 List 或 Java 14+ `record`（只读数据载体）
  - 不同点：Python tuple 常用于**组包/解包**、函数多返回值；Java 无直接语法

| 特性 | tuple | list |
|------|-------|------|
| 可变 | ❌ | ✅ |
| 有序 | ✅ | ✅ |
| 可重复 | ✅ | ✅ |
| 语法 | `(1, 2, 3)` 或 `1, 2, 3` | `[1, 2, 3]` |

### 2. 基本操作

```python
t1 = (1, 234, 2, 43, 23, '456', 2)

# 索引和切片 — 和 list 完全一样
print(t1[1])        # 234
print(t1[1::2])     # (234, 43, '456')

# 只有两个方法（因为没有增删改）
print(t1.count(2))  # 2（元素 2 出现了 2 次）
print(t1.index(2))  # 2（元素 2 第一次出现的索引）

# 定义空元组和单元素元组
t2 = ()
t3 = (100,)   # ⚠️ 必须加逗号！(100) 只是带括号的 int
```

> ⚠️ **关键坑**：`(100)` 是 int，`(100,)` 才是 tuple。逗号是元组的灵魂，不是括号。

### 3. 组包与解包（tuple 最常用场景）

```python
# 组包 — 把多个值打包成元组（括号可省略）
t = 12, 213, 321, 124, 52   # 自动组包为 (12, 213, 321, 124, 52)

# 基础解包 — 一对一拆开（数量必须匹配）
a, b, c, d, e = t
print(a, b)  # 12 213

# 扩展解包 — 用 * 收集剩余（组成 list）
first, second, *others = t
print(first)   # 12
print(others)  # [321, 124, 52] ← * 收集的变成 list

*head, last = t
print(head)    # [12, 213, 321, 124]
print(last)    # 52

# 经典应用：一行交换变量（背后就是元组组包+解包）
a, b = 10, 20
a, b = b, a         # b, a 先组包 → 再解包给 a, b
print(a, b)         # 20 10

# 三变量轮转
a, b, c = 100, 200, 300
c, a, b = a, b, c   # 轮转赋值
```

### 4. 实战：遍历嵌套元组

```python
students = (
    ("S001", "王林", 85, 78, 92),
    ("S002", "李慕婉", 92, 88, 95),
    ("S003", "十三", 78, 82, 85),
)

# 解包遍历 — 一行拿到所有字段
for sid, name, chinese, math, english in students:
    total = chinese + math + english
    avg = total / 3
    print(f"{sid} {name} 总分:{total} 均分:{avg:.1f}")
```

## set 集合

> 📺 **对应课程**：第 49–51 集

### 1. 什么是 set

- **是什么**：集合像一袋弹珠——每颗弹珠颜色不同（自动去重），倒出来顺序不固定（无序）。
- **Java 对比**：≈ `HashSet`，去重、无序、数学集合运算

| | Python set | Java HashSet |
|------|-----------|-------------|
| 定义 | `{1, 2, 3}` | `new HashSet<>(Set.of(1,2,3))` |
| 空集合 | `set()` **不是 `{}`** | `new HashSet<>()` |
| 交并差 | `&` `\|` `-` 运算符 | `retainAll/addAll/removeAll` |

> ⚠️ `{}` 是空**字典**，不是空集合！空集合必须写 `set()`。

### 2. 基本操作

```python
s = {1, 2, 3, 4, 5, 2, 4}  # 重复自动去除
print(s)  # {1, 2, 3, 4, 5}（顺序不定）

s.add(800)       # 添加元素
s.remove(100)    # 删除元素（不存在会报错）
s.pop()          # 随机删除一个并返回
s.clear()        # 清空
```

### 3. 集合运算（核心优势）

```python
s1 = {'A', 'B', 'C', 'D', 'E', 'F'}
s2 = {'B', 'C', 'G', 'H', 'I', 'J'}

# 交集（两者都有）
print(s1 & s2)                    # {'B', 'C'}
print(s1.intersection(s2))        # 方法版

# 并集（合并去重）
print(s1 | s2)                    # {'A','B','C','D','E','F','G','H','I','J'}
print(s1.union(s2))

# 差集（只在 s1 不在 s2）
print(s1 - s2)                    # {'A', 'D', 'E', 'F'}
print(s1.difference(s2))
```

### 4. 集合推导式

```python
# 语法：{表达式 for 元素 in 集合 if 条件}
football = {"王林", "韩立", "厉飞雨", "紫灵"}
basketball = {"张铁", "王林", "韩立"}

# 只踢足球不打篮球的学生
result = {s for s in football if s not in basketball}
```

## dict 字典

> 📺 **对应课程**：第 52–55 集

### 1. 什么是 dict

- **是什么**：字典像真正的词典——根据词条（key）查解释（value），而不是一页页翻。
- **Java 对比**：≈ `HashMap`，键值对、O(1) 查找

| | Python dict | Java HashMap |
|------|------------|-------------|
| 定义 | `{"a": 1, "b": 2}` | `new HashMap<>(Map.of("a",1,"b",2))` |
| 取值 | `d["a"]` 或 `d.get("a")` | `map.get("a")` |
| 键类型 | 必须可哈希（str/int/tuple） | 必须实现 hashCode/equals |
| 有序性 | ✅ 3.7+ 插入有序 | ❌（除非 LinkedHashMap） |

> ⚠️ **key 不能重复**，重复赋值会覆盖旧值。key 必须是**不可变类型**（str/int/float/tuple），不能是 list/set/dict。

### 2. 增删改查

```python
d = {"mosifu": 688, "jianl": 334, "mo": 666}

# 增/改 — 同一语法
d["你好"] = 350     # key 不存在 = 添加
d["你好"] = 600     # key 存在 = 修改

# 查
print(d["mosifu"])      # 688（key 不存在报 KeyError）
print(d.get("mosifu"))  # 688（key 不存在返回 None，更安全）

# 删
d.pop("mo")            # 删除并返回 value
del d["jianl"]         # 纯删除

# 获取所有 key / value / 键值对
print(d.keys())    # dict_keys(['mosifu', '你好'])
print(d.values())  # dict_values([688, 600])
print(d.items())   # dict_items([('mosifu', 688), ('你好', 600)])
```

### 3. 遍历字典

```python
# 方式1：遍历 key
for k in d.keys():
    print(f"{k}: {d[k]}")

# 方式2：遍历 key+value（推荐！）
for k, v in d.items():
    print(f"{k}: {v}")
```

### 4. 嵌套字典（重要！）

```python
# 结构：{key: {子key: 值, ...}, ...}
shop = {
    "苹果": {"price": 5.5, "num": 10},
    "香蕉": {"price": 3.0, "num": 20},
}

# 访问嵌套值
print(shop["苹果"]["price"])  # 5.5

# 遍历嵌套dict
for name, info in shop.items():
    print(f"{name}: 单价{info['price']}, 库存{info['num']}")
```

> ⚠️ 嵌套字典访问用 `d["k1"]["k2"]` 时注意引号冲突：外层 f-string 用了双引号，内层 `["key"]` 就要用单引号。

### 5. 综合案例

> 💡 以下两个案例来自学习者的实战练习，串联了 dict + match + while + input 等全部已学知识。

**案例1：购物车系统**（增删改查 + 嵌套字典 + match-case）：

```python
# 制作菜单
menu = """
######## 欢迎来到购物车系统 ###########
#           1.添加购物车            #
#           2.修改购物车            #
#           3.删除购物车            #
#           4.查询购物车            #
#           5.退出购物车            #
###################################
"""
shop_dict = {}
while True:
    print(menu)
    op = input("请输入需要执行的操作数字(1-5): ")
    match op:
        case '1':   # 添加
            name = input("请输入需要录入的商品名称: ")
            if name in shop_dict:
                print(f"{name}该商品已存在！")
                continue
            price = float(input("请输入需要录入的商品价格: "))
            num = int(input("请输入需要录入的商品数量: "))
            shop_dict[name] = {'price': price, 'num': num}
            print(f"添加商品成功: {name}, 价格: {price}, 数量: {num}")

        case '2':   # 修改
            name = input("请输入需要修改的商品名称: ")
            if name not in shop_dict:
                print(f"{name}该商品不存在！")
                continue
            price = float(input("请输入需要修改的商品价格: "))
            num = int(input("请输入需要修改的商品数量: "))
            shop_dict[name] = {'price': price, 'num': num}
            print(f"修改商品成功: {name}, 价格: {price}, 数量: {num}")

        case '3':   # 删除
            name = input("请输入需要删除的商品名称: ")
            if name not in shop_dict:
                print(f"{name}该商品不存在！")
            else:
                shop_dict.pop(name)
                print(f"删除{name}商品成功")

        case '4':   # 查询
            print("当前购物车中商品信息为:")
            for k, v in shop_dict.items():
                print(f"商品名称: {k}, 商品价格: {v['price']}, 商品数量: {v['num']}")

        case '5':   # 退出
            break

        case _:
            print("未知操作，请重试")
```

**购物车存储结构**：`{'商品名': {'price': 单价, 'num': 数量}, ...}`

**案例2：教务管理系统**（增删改查 + 统计排名）：

```python
menu = """
[菜单]
1.添加学生信息  2.修改学生信息  3.删除学生信息
4.查询学生信息  5.列出所有信息  6.统计班级成绩  7.退出系统
"""
stu_mes = {}
while True:
    print(menu)
    op = input("请输入需要进行的操作(1-7): ")
    match op:
        case '1':   # 添加学生信息
            print("添加学生信息:")
            name = input("请输入学生姓名: ")
            if name in stu_mes:
                print(f"{name}该学生已存在！")
                continue
            chinese = float(input("请输入学生语文成绩: "))
            math = float(input("请输入学生数学成绩: "))
            english = float(input("请输入学生英语成绩: "))
            stu_mes[name] = {'chinese': chinese, 'math': math, 'english': english}
            print(f"添加{name}学生信息成功！语文{chinese}分, 数学{math}分, 英语{english}分")

        case '2':   # 修改学生信息
            print("修改学生信息:")
            name = input("请输入需要修改信息的学生姓名: ")
            if name not in stu_mes:
                print(f"{name}该学生不存在！")
                continue
            chinese = float(input("需要修改的语文成绩: "))
            math = float(input("需要修改的数学成绩: "))
            english = float(input("需要修改的英语成绩: "))
            stu_mes[name] = {'chinese': chinese, 'math': math, 'english': english}
            print(f"修改{name}学生信息成功！")

        case '3':   # 删除学生信息
            print("删除学生信息:")
            name = input("请输入需要删除信息的学生姓名: ")
            if name not in stu_mes:
                print(f"{name}该学生不存在！")
                continue
            del stu_mes[name]
            print(f"删除{name}学生的信息成功")

        case '4':   # 查询学生信息
            print("查询学生信息:")
            name = input("请输入需要查询的学生姓名: ")
            if name not in stu_mes:
                print(f"{name}该学生不存在！")
                continue
            s = stu_mes[name]
            print(f"查询{name}学生信息成功！语文{s['chinese']}分, 数学{s['math']}分, 英语{s['english']}分")

        case '5':   # 列出所有学生
            print("查询所有学生信息:")
            for k, v in stu_mes.items():
                print(f"学生{k}的成绩为: 语文:{v['chinese']}, 数学:{v['math']}, 英语:{v['english']}")

        case '6':   # 统计班级成绩
            print("统计全班信息:")

            # 语文统计
            chinese_all = sum(stu_mes[k]['chinese'] for k in stu_mes)
            chinese_avg = chinese_all / len(stu_mes)
            chinese_min = min(stu_mes[k]['chinese'] for k in stu_mes)
            chinese_max = max(stu_mes[k]['chinese'] for k in stu_mes)
            ch_min_stu = [n for n in stu_mes if stu_mes[n]['chinese'] == chinese_min]
            ch_max_stu = [n for n in stu_mes if stu_mes[n]['chinese'] == chinese_max]
            print(f"语文平均分: {chinese_avg:.1f}; 最高: {chinese_max}{ch_max_stu}; 最低: {chinese_min}{ch_min_stu}")

            # 数学统计
            math_all = sum(stu_mes[k]['math'] for k in stu_mes)
            math_avg = math_all / len(stu_mes)
            math_min = min(stu_mes[k]['math'] for k in stu_mes)
            math_max = max(stu_mes[k]['math'] for k in stu_mes)
            m_min_stu = [n for n in stu_mes if stu_mes[n]['math'] == math_min]
            m_max_stu = [n for n in stu_mes if stu_mes[n]['math'] == math_max]
            print(f"数学平均分: {math_avg:.1f}; 最高: {math_max}{m_max_stu}; 最低: {math_min}{m_min_stu}")

            # 英语统计
            english_all = sum(stu_mes[k]['english'] for k in stu_mes)
            english_avg = english_all / len(stu_mes)
            english_min = min(stu_mes[k]['english'] for k in stu_mes)
            english_max = max(stu_mes[k]['english'] for k in stu_mes)
            e_min_stu = [n for n in stu_mes if stu_mes[n]['english'] == english_min]
            e_max_stu = [n for n in stu_mes if stu_mes[n]['english'] == english_max]
            print(f"英语平均分: {english_avg:.1f}; 最高: {english_max}{e_max_stu}; 最低: {english_min}{e_min_stu}")

        case '7':   # 退出
            break

        case _:
            print("非法操作！")
```

**教务系统存储结构**：`{'姓名': {'chinese': 分, 'math': 分, 'english': 分}, ...}`
**亮点**：统计科目前三名用列表推导式 `[n for n in stu_mes if stu_mes[n]['chinese'] == chinese_max]` 支持并列排名

## 四大容器总对比

| | list | tuple | set | dict |
|------|:--:|:--:|:--:|:--:|
| 符号 | `[]` | `()` | `{}` / `set()` | `{k:v}` |
| 有序 | ✅ | ✅ | ❌ | ✅(3.7+) |
| 可变 | ✅ | ❌ | ✅ | ✅ |
| 可重复 | ✅ | ✅ | ❌ | key 不重复 |
| 索引 | ✅ | ✅ | ❌ | ✅(key) |
| 典型场景 | 有序集合 | 不变数据/解包 | 去重/交并差 | 键值映射 |

---

## 常见坑点（下篇：tuple/set/dict）

7. **`(100)` vs `(100,)`** — 单元素元组的逗号不能省
8. **`{}` 是空 dict，不是空 set** — 空集合必须 `set()`
9. **dict key 不可变** — list/set/dict 不能作 key
10. **f-string 嵌套引号** — `f"...{d['key']}..."` 双引号冲突，外层或内层换单引号
11. **set 的 `remove()` vs `discard()`** — remove 元素不存在报错，discard 不报错
12. **dict 的 `pop(key)` vs `del d[key]`** — pop 返回被删值，del 不返回

---

## 小结

1. **tuple**：不可变 list，灵魂是逗号（单元素必加逗号）；组包/解包是日常操作
2. **set**：自动去重 + 交并差运算；空集合用 `set()`；集合推导式 `{x for x in ...}`
3. **dict**：键值映射结构；`d.get()` 比 `d[]` 安全；嵌套 dict 注意 f-string 引号冲突
