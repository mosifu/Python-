## list 列表

> 📺 **对应课程**：第 36–42 集

### 1. 什么是 list

- **是什么**：列表就像火车 — 每节车厢可以装不同类型的东西，车厢编号从 0 开始。
- **Java 对比**：
  - 相似点：≈ `ArrayList`，有序、可变、可索引
  - 不同点：Python 列表**可存不同类型**；语法直接用 `[]`，无需 `new`

```python
# 定义 — 比 Java 简洁得多
s = [1, 3, 4.5, 'hello', True]   # 混合类型完全 OK
print(type(s))  # <class 'list'>
```

| | Python list | Java ArrayList |
|------|-------------|---------------|
| 定义 | `[1, 2, 3]` | `new ArrayList<>(List.of(1,2,3))` |
| 混合类型 | ✅ 可以 | ❌ 必须同类型（泛型） |
| 索引 | `s[0]` `s[-1]` | `s.get(0)` |
| 修改 | `s[0] = 10` | `s.set(0, 10)` |

### 2. 索引与切片

```python
h = ["A", "B", "C", "D", "E", "F", "G", "H", "I", "J"]

# 索引 — 正向从 0，反向从 -1
print(h[0])    # A
print(h[-1])   # J（最后一个）
print(h[-5])   # F（倒数第5个）

# 切片 — list[start:end:step]
print(h[0:6])     # ['A','B','C','D','E','F']  end不包含
print(h[:4])      # ['A','B','C','D']  省略start=从头
print(h[1:-1:2])  # ['B','D','F','H']  步长为2
print(h[::-1])    # ['J','I',...,'A']  step=-1=反转
```

> 💡 切片规则和 `range()` 一样：**不包含 end**，支持反向步长。

### 3. 增删改查

| 操作 | 方法 | 说明 | Java 类比 |
|------|------|------|----------|
| **追加** | `.append(78)` | 末尾加一个元素 | `list.add(78)` |
| **插入** | `.insert(1, 78)` | 在索引1前插入 | `list.add(1, 78)` |
| **删除(值)** | `.remove(78)` | 删第一个匹配值 | `list.remove(Integer.valueOf(78))` |
| **删除(索引)** | `.pop(1)` | 删索引元素并返回 | `list.remove(1)` |
| **删除(索引)** | `del s[2]` | 纯删除，不返回 | 无直接对应 |
| **排序** | `.sort()` | 原地排序 | `Collections.sort(list)` |
| **反转** | `.reverse()` | 原地反转 | `Collections.reverse(list)` |
| **查索引** | `.index(x)` | 找第一个匹配 | `list.indexOf(x)` |
| **统计** | `.count(x)` | 计数 | `Collections.frequency(list, x)` |

```python
g = [56, 234, 634, 23, 63, 74, 211, 5]

g.append(78)       # [56, 234, 634, 23, 63, 74, 211, 5, 78]
g.insert(1, 99)    # [56, 99, 234, ..., 78]    在索引1前插入
g.remove(78)       # [56, 99, 234, ..., 5]     删第一个78
g.pop(1)           # [56, 234, ..., 5]         删索引1
g.pop()            # [56, 234, ..., 211]       无参数=删最后一个
g.sort()           # [5, 23, 56, 63, 74, 211, 234]
g.reverse()        # [234, 211, 74, 63, 56, 23, 5]
```

> ⚠️ `.sort()` 要求**元素类型相同**（不能混排 str 和 int）。
> ⚠️ `.sort()` 是**原地修改**返回 `None`，不是返回新列表。

### 4. 列表合并

```python
# 方式1：+ 拼接（Java 也可以用 addAll）
new_list = [1, 2, 3] + [4, 5, 6]     # [1, 2, 3, 4, 5, 6]

# 方式2：解包（Python 独有）
new_list = [*list1, *list2]

# 方式3：extend
list1.extend(list2)
```

### 5. 列表去重

```python
# 经典去重法：遍历 + not in
origin = [12, 24, 35, 12, 535, 125, 3, 24]
unique = []
for item in origin:
    if item not in unique:
        unique.append(item)

# 快捷法：set 去重（学到 set 时会讲）
unique = list(set(origin))  # 但会打乱顺序
```

### 6. 列表推导式（List Comprehension）

- **是什么**：用一行代码生成列表，就像数学里的集合推导 `{x² | x∈N, x<10}`。
- **Java 对比**：≈ Stream API `map().filter().collect()`

```python
# 语法1：[表达式 for i in 序列]
squares = [i**2 for i in range(1, 21)]     # 1-20的平方
# Java: IntStream.range(1,21).map(i->i*i).boxed().collect(toList())

# 语法2：[表达式 for i in 序列 if 条件]
evens = [i for i in range(1, 31) if i % 2 == 0]          # 偶数
squares_even = [i**2 for i in range(1, 31) if i % 2 == 0] # 偶数的平方

# 复杂筛选：能被3或5整除的数的平方
result = [i**2 for i in range(1, 31) if i % 3 == 0 or i % 5 == 0]

# 提取正数
positives = [i for i in [11, -21, 1231, -34, 10] if i > 0]
```

## str 字符串操作

> 📺 **对应课程**：第 43–46 集

### 1. str 的三大特性

| 特性 | 含义 | Java 对比 |
|------|------|----------|
| **不可变** | 一旦创建，内容不能修改 | ≈ `String` 的不可变性 |
| **有序** | 有索引，可切片 | ✅ 一样 |
| **可迭代** | 能用 `for` 遍历 | ✅ 一样 |

```python
s = "Hello-Python"
# s[0] = "h"       # ❌ TypeError: str 不可修改！

s = s.replace("H", "h")  # ✅ 返回新字符串，原字符串不变
```

> 💡 Python 无 `char` 类型，单个字符 `'a'` 也是 `str`。

### 2. 索引与切片

```python
str1 = "Hello-Python"

print(str1[0])       # H
print(str1[-1])      # n
print(str1[0:5])     # Hello
print(str1[::2])     # HloPto（隔一个取一个）
print(str1[::-1])    # nohtyP-olleH（反转）

# 负步长切片 — 开始索引必须在结束索引**右边**
print(str1[-8:-13:-1])  # oP-oll  从-8倒着走到-13
# print(str1[0:5:-1])   # 空！start=0, end=5, step=-1 → 方向不对
```

> ⚠️ **负步长坑**：`step` 为负时，start 必须在 end **右边**（索引更大的一方），否则结果为空。

### 3. 常用方法速查表

| 方法 | 作用 | 示例 | Java 类比 |
|------|------|------|----------|
| `.find("x")` | 找第一个匹配索引，没找到返回 -1 | `"hello".find("e")` → 1 | `indexOf()` |
| `.count("x")` | 统计出现次数 | `"hello".count("l")` → 2 | 用循环统计 |
| `.split("-")` | 按分隔符切割成**列表** | `"a-b-c".split("-")` → `['a','b','c']` | `split()` |
| `.upper()` | 全转大写 | `"hello".upper()` → `"HELLO"` | `toUpperCase()` |
| `.lower()` | 全转小写 | `"Hello".lower()` → `"hello"` | `toLowerCase()` |
| `.strip()` | 去两端空格 | `" hi ".strip()` → `"hi"` | `trim()` |
| `.replace(a, b)` | 替换子串 | `"a-b".replace("-", "_")` | `replace()` |
| `.startswith("x")` | 以 x 开头？ | `"hi".startswith("h")` → True | `startsWith()` |
| `.endswith("x")` | 以 x 结尾？ | `"hi".endswith("i")` → True | `endsWith()` |

```python
str2 = "  Hello-Python-Hello-World  "

print(str2.find("Hello"))     # 2（第一个Hello的位置）
print(str2.count("Hello"))    # 2（出现了2次）
print(str2.split('-'))        # ['  Hello', 'Python', 'Hello', 'World  ']
print(str2.upper())           # '  HELLO-PYTHON-HELLO-WORLD  '
print(str2.strip())           # 'Hello-Python-Hello-World'
print(str2.replace("-", "_")) # '  Hello_Python_Hello_World  '
print(str2)                   # 原始字符串没变！(不可变性)
```

### 4. 实战案例

```python
# 案例1：邮箱格式验证
email = input("请输入邮箱：")
if email.count("@") == 1 and "." in email:
    print("邮箱格式正确")
else:
    print("邮箱格式错误")

# 案例2：回文判断（正反读一样）
text = input("请输入字符串：")
half = len(text) // 2          # 整除取一半长度
if text[:half] == text[-1:-half-1:-1]:
    print(f"'{text}' 是回文！")
else:
    print(f"'{text}' 不是回文")

# 案例3：反转 + 大写
text = "hello"
reversed_upper = text[::-1].upper()  # OLLEH
```

## 常见坑点

1. **`.sort()` 返回 None** — 它是原地排序，别写 `new = list.sort()`
2. **list 越界** — 修改索引超出范围报 `IndexError`，切片越界不报错只返回空
3. **str 不可修改** — `s[0] = 'x'` 报 `TypeError`，用 `replace()` 或切片拼接
4. **`%` 和 `//` 混淆** — `%` 取余、`//` 整除，回文判断用 `//`
5. **负步长方向** — `[0:5:-1]` 返回空，start 必须在 end 右边
6. **列表推导式可读性** — 超过 2 层嵌套或复杂条件时，拆成普通 for 循环更清晰

---

## 小结

1. **list**：有序可变容器，索引 `[0]`/`[-1]`，切片 `[:]`，常用 `append/sort/pop`，推导式 `[x for x in list if]` 替代 for+append
2. **str**：有序不可变序列，`find/count/split/upper/strip/replace` 返回新值，原字符串不变
3. 切片三要素 `[start:end:step]`，end 不包含，负步长需方向一致
