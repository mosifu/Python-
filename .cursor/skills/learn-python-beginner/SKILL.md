## 教学原则

1. 一次只讲一个概念，讲完再进入下一个
2. 用生活化类比解释抽象概念
3. 每个知识点配 1 个可运行的小例子
4. 代码注释用中文，术语首次出现附英文
5. 不堆砌与当前课时无关的高级特性
6. **Java 类比原则**：用户具备 Java 基础时，每个新知识点优先用 Java 中熟悉的概念做类比，并明确列出 **相似点** 与 **不同点**，帮助快速建立知识映射、避免把 Java 习惯错误套用到 Python

### Java 类比通用对照（讲解时按需引用）

| Python 概念 | Java 对应 | 相似点 | 不同点 |
|------------|----------|--------|--------|
| 解释型运行 | JVM 编译后运行 | 都有变量、函数、类、异常 | Python 无需 `main` 入口即可运行脚本；无 `public static void main` |
| 缩进 | `{}` 花括号 | 都表达代码块层级 | Python **用缩进** 表示块，无分号要求 |
| `list` | `ArrayList` | 有序、可变、可索引 | Python 列表可存不同类型；语法更简洁 `[1, 2, 3]` |
| `tuple` | 无直接对应（近似不可变列表） | 有序、可索引 | **不可变**；常用于多返回值 |
| `dict` | `HashMap` | 键值对、O(1) 查找 | Python 3.7+ dict 有序；键必须可哈希 |
| `set` | `HashSet` | 去重、无序集合 | Python set 用 `{}` 或 `set()` |
| `def` 函数 | 方法/静态方法 | 参数、返回值、作用域 | 函数是一等公民，可赋值、当参数传递 |
| `class` / `self` | `class` / `this` | 类、对象、继承、多态 | Python 用 `self` 显式传参；支持多继承；鸭子类型 |
| `try/except` | `try/catch` | 异常捕获与传播 | Python 用 `except`，`finally` 相同 |
| `import` | `import` / `package` | 模块化复用 | Python 模块即 `.py` 文件，无 `pom.xml` |
| `None` | `null` | 表示空值 | `None` 是单例对象，判断用 `is None` |
| `match` | `switch`（Java 14+） | 模式匹配分支 | Python 3.10+ `match/case`，功能更灵活 |
| `lambda` | Lambda 表达式 | 匿名函数 | Python lambda 只能写单行表达式 |
| 类型注解 | 泛型/类型声明 | 提升可读性 | Python 注解**不强制**，运行时仍是动态类型 |

### 阶段 2：Python 核心语法

#### 2.1 数据存储与运算

**集数**：09–21
**内容**：字面量、变量、标识符、数据类型、字符串（定义/拼接/格式化）、输入输出、算术/赋值/比较/逻辑运算符

**Java 类比重点**：
- `int`/`float`/`str`/`bool` ≈ Java 基本类型与 `String`、`boolean`
- **不同**：Python 变量无需声明类型；`input()` 返回的永远是字符串
- 字符串格式化：`f"{name}"` ≈ `String.format()` / 文本块

**里程碑练习**：用户信息输入与格式化输出小程序

---

#### 2.2 流程控制语句

**集数**：22–35
**内容**：`if`/`elif`/`else`、`match` 模式匹配、`while`、`for`、`range`、嵌套循环、`break`/`continue`、综合案例（猜数字）

**Java 类比重点**：
- `if/elif/else` ≈ `if/else if/else`
- `while`/`for` 逻辑相同
- **不同**：`for item in list` ≈ 增强 for；`match/case` ≈ `switch` 但更强大
- Python 无 `do-while`

**里程碑练习**：猜数字游戏（对应课程 35 集）

---

#### 2.3 数据容器

**集数**：36–56
**内容**：`list`、`str` 操作、`tuple`（组包/解包）、`set`、`dict`、推导式、综合案例

**Java 类比重点**：

| Python | Java |
|--------|------|
| `list` | `ArrayList` |
| `tuple` | 不可变数据（可用 `record` 类比） |
| `set` | `HashSet` |
| `dict` | `HashMap` |
| 列表推导 `[x*2 for x in arr]` | Stream `map().collect()` |

- **不同**：切片 `list[1:3]` Java 无原生语法；tuple 常用于函数多返回值

**里程碑练习**：用 dict 实现简易通讯录（对应 54–55 集案例）

---

#### 2.4 函数

**集数**：57–70
**内容**：函数定义、参数与返回值、嵌套调用、作用域、传参方式、默认参数、不定长参数（`*args`/`**kwargs`）、函数作参数、`lambda`、递归

**Java 类比重点**：
- 方法定义 ≈ `def`，但 Python 函数独立于类
- 默认参数 ≈ 方法重载的简化替代
- `*args` ≈ 可变参数 `String... args`
- **不同**：Python 支持函数作为参数传递；`lambda` 限制更多

**里程碑练习**：实现带默认参数和可变参数的统计函数

---

#### 2.5 类型注解与模块

**集数**：71–76
**内容**：类型注解、函数类型注解、`import`、自定义模块、`package`

**Java 类比重点**：
- 类型注解 ≈ Java 显式类型，但**仅提示，不强制**
- `import module` ≈ `import com.xxx.Class`
- `package` 目录结构 ≈ Java package 目录
- **不同**：Python 用 `__init__.py`（3.3+ 命名空间包可省略）

---

#### 2.6 面向对象基础

**集数**：77–85
**内容**：类与对象、实例方法、魔法方法（`__init__` 等）、实例属性与类属性、教务系统综合案例

**Java 类比重点**：
- `class Student:` ≈ `class Student { }`
- `__init__` ≈ 构造方法
- `self` ≈ `this`（必须显式写在参数里）
- **不同**：无 `public/private` 关键字，用 `_` 约定私有；魔法方法如 `__str__` ≈ `toString()`

**里程碑练习**：教务系统案例（添加/修改/删除/查询学生）

---

#### 2.7 异常处理

**集数**：86–87
**内容**：异常机制、案例代码完善

**Java 类比重点**：
- `try/except/finally` ≈ `try/catch/finally`
- `raise` ≈ `throw`
- **不同**：Python 异常类不必声明 `throws`

**阶段 2 总结**：完成核心语法后，已具备与 Java SE 基础对等的能力。

## 「生成笔记」命令

当用户发送 **「生成笔记」** 时，执行以下流程：

### 1. 确定章节归属
- 判断当前讲解内容属于哪个阶段（阶段 1–6）和哪个章节（如 2.3 数据容器）

### 2. 生成 Markdown 知识文档
在当前工作目录下创建 `notes/` 目录，按以下结构组织：

```
./notes/
├── 阶段1-Python启航/
│   ├── 01-初识Python.md
│   └── ...
├── 阶段2-核心语法/
│   ├── 2.1-数据存储与运算.md
│   ├── 2.2-流程控制.md
│   ├── 2.3-数据容器.md
│   ├── 2.4-函数.md
│   ├── 2.5-类型注解与模块.md
│   ├── 2.6-面向对象基础.md
│   └── 2.7-异常处理.md

├── 阶段4-网络爬虫/
```

在 `./notes/阶段X-章节名/` 对应目录下创建 `examples/` 子目录，存放可运行的 `.py` 示例文件：

- 文件名用数字前缀排序
